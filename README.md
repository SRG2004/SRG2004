### 🔥 Featured Projects

#### 🛡️ [CyberGuard AI](https://github.com/SRG2004) — Phishing & Threat Detection Platform
`React` `Node.js` `XGBoost` `Chrome MV3`
ML detection service (XGBoost, 39 features, 150K+ URLs from PhishTank/OpenPhish/URLhaus) at 99.1% accuracy, plus a TF-IDF + LinearSVC email-phishing classifier at 99.2% accuracy. Integrated into a Chrome MV3 extension with a Node.js/Express/MongoDB backend for real-time scanning.

#### 📰 [DailyNewsAI](https://github.com/SRG2004) — AI News Monitoring & Fake-News Detection
`Python` `FastAPI` `React`
5-layer pipeline (ingestion → NLP cleaning → classification → dashboard → bot) combining TF-IDF + Logistic Regression, a fine-tuned DistilBERT model, and LDA topic modeling — exposed via FastAPI + React.

#### 🌊 [Tarang](https://github.com/SRG2004) — Ocean Hazard Response Platform · *SIH 2025 Finalist*
`React` `Firebase` `Gemini AI`
Disaster-management platform built for INCOIS enabling citizen hazard reporting with Gemini-based auto-verification and authority/field-team coordination, on Firebase Functions/Firestore with a multilingual React Native app.

#### 🌾 [AgriChain (TerraFlow)](https://github.com/SRG2004) — Agri Supply Chain Traceability
`MERN` `JWT` `Solidity`
Role-based MERN application (Farmer/Distributor/Retailer/Consumer) with JWT auth, MongoDB data modeling, QR-code product traceability, and a Solidity/Truffle smart-contract layer.

#### 🌱 [Crop Recommendation System](https://github.com/SRG2004)
`Python` `Flask` `Scikit-learn`
Benchmarked Random Forest, Decision Tree, KNN, and Logistic Regression on soil/climate data; deployed the top model (>99% accuracy) via Flask with EDA visualizations for 20+ crop recommendations.

#### 📸 [Face Recognition Attendance System](https://github.com/SRG2004)
`Python` `OpenCV` `SQLite`
Tkinter desktop app using dlib-based face_recognition for automated attendance (~80% less manual effort), with a CRUD interface and one-click Excel export.
